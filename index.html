Admin


Admin

As an Admin, users are able to add, delete, or edit values in the database.



Students

Students


Users are able to modify students by navigating to Students.

  • To add a student click the Add Student button at the top of the page.
  • A student can be searched by either their last name, first name, or both.
  • Documents for a student can be uploaded by clicking the View/Edit button. Forms can be sent to be signed as well.
  • If a student needs to be deleted click the red Delete button. Note this action cannot be undone.

Schools

Schools

Users are able to modify and view details of a school by navigating to Schools.

  • To add a school click the Add School button at the top of the page.
  • A school can be searched by either its short name, full name, or both.
  • Forms for a school can be created by clicking the View forms button.
  • Meetings for a school can be created and viewed by clicking the View Meetings button.
  • If a school needs to be removed click the red Delete button. Note this action cannot be undone.
